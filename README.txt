LFJC Online Exam System — GitHub Pages Version
=================================================

How to Host (Free, Permanent):
------------------------------
1. Go to https://github.com and log in.
2. Click "+" → "New Repository".
3. Name it: LFJC_Online_Exam (Public).
4. After creation, click "Add file" → "Upload files".
5. Upload 'index.html' from this folder.
6. Click "Commit changes".
7. Go to Settings → Pages →
   • Source: Deploy from a branch
   • Branch: main (or master)
   • Folder: /(root)
   • Save
8. Wait 1–2 minutes; your live link will appear as:
   https://YOURUSERNAME.github.io/LFJC_Online_Exam/

Student Access:
---------------
Share the above link with students. They can open it on phones, tablets, or computers.
